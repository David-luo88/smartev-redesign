# SmartEV Redesign

Static one-page interactive Smart EV Platform presentation.

## Local Preview

```bash
npm run dev
```

Open `http://localhost:4173`.

## Build Check

```bash
npm run build
```

## Render

This repository includes `render.yaml` for a static web service.

- Runtime: Static
- Publish path: `.`
- Rewrite: `/* -> /index.html`
